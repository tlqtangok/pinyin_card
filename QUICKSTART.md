# 🎉 QUICK START GUIDE - 拼音乐园

## For Windows Users (Easiest Way!)

### Step 1: Open This Folder
You're already here! ✅

### Step 2: Start the App
**Double-click** `start.bat` file

### Step 3: Open Browser
The app will automatically tell you to visit:
```
http://localhost:8888
```

### Step 4: Have Fun!
- Click any colorful card to hear the pinyin pronunciation
- Use the letter buttons to filter (a, b, c, d...)
- Press 'R' key for a random sound!
- Notice how each group of 4 cards (4 tones) is on one line!

---

## Manual Start (Alternative)

If you prefer using the command line:

```bash
# Install dependencies (first time only)
npm install

# Start server (default port 3000)
npm start

# OR specify a custom port
set PORT=8888
node server.js
```

## 📱 Usage Tips for Parents

1. **Start Simple**: Begin with letter 'A' or 'B' filter
2. **One Tone at a Time**: Focus on learning 一声 (tone 1) first
3. **Daily Practice**: 10 minutes per day is perfect
4. **Make it a Game**: 
   - "Can you find all the '一声' sounds?"
   - "Let's click 10 different cards!"
   - "Can you repeat what you hear?"
5. **Progress Gradually**: A → B → C → all letters

## 🔧 Troubleshooting

**Q: Browser can't connect?**
- Make sure `start.bat` is running
- Look for "Pinyin Kids App is running" message
- Check that port 8888 is not in use

**Q: No sound?**
- Check computer volume is on
- Try clicking another card
- Check browser console (F12) for errors

**Q: Page looks broken?**
- Try refreshing browser (F5)
- Clear browser cache (Ctrl+Shift+Delete)

**Q: Want to use different port?**
- Edit `start.bat` file
- Change `set PORT=8888` to your preferred port

## 🎓 Learning Path for Your Daughter

### Week 1-2: Basic Sounds
- Filter by 'A' and listen to a1, a2, a3, a4
- Filter by 'B' and explore ba, bi, bo, bu sounds
- Practice repeating after hearing

### Week 3-4: More Consonants  
- Try C, D, F, G sounds
- Mix and match - click randomly
- Start recognizing tone differences

### Month 2: Advanced Practice
- Use "All" view and explore everything
- Play the "Random Sound" game (press 'R')
- Try to predict the tone before clicking

### Ongoing: Daily Fun
- Make it part of daily routine
- Celebrate progress with stickers/rewards
- Keep it fun and pressure-free!
